2 집단추출법
해설
표본 추출 방법에는 다양한 기법이 존재하지만, 집단추출법은 통계학적으로 정식 용어가 아니며 실제 사용되는 표본 추출 방법에 해당하지 않는다. 주어진 선택지 중 층화추출법, 집락추출법, 단순무작위추출법은 모두 실무 및 이론에서 사용되는 표본 추출 방법이다.

비기봇 해설
표본추출법은 모집단에서 대표성을 유지하면서 일부를 선택하는 방법을 의미합니다. 대표적인 표본추출법에는 단순무작위추출법, 층화추출법, 집락추출법 등이 있습니다. 그러나 ‘집단추출법’이라는 용어는 공식적인 표본추출 방법에 포함되지 않습니다.

1. 층화추출법
: 모집단을 몇 개의 층(집단)으로 나누고, 각 층에서 일정한 비율로 표본을 추출하는 방법입니다. 예를 들어, 남녀 성별로 나누어 표본을 선정하는 경우가 이에 해당합니다. 층별로 대표적인 표본을 뽑기 때문에 모집단의 다양한 특성을 반영할 수 있습니다.

2. 집단추출법
: ‘집단추출법’이라는 용어는 공식적인 표본추출법에 포함되지 않습니다. 실제로 사용되는 표본추출법 중에는 ‘집락추출법’이 있지만, 집단추출법이라는 용어는 존재하지 않으므로 오답입니다.

3. 집락추출법
: 모집단을 몇 개의 군집(집락)으로 나누고, 일부 집락을 무작위로 선택한 뒤 해당 집락 내의 모든 개체를 표본으로 선정하는 방법입니다. 예를 들어, 학교별로 집락을 설정한 후 몇 개 학교를 무작위로 선택하여 해당 학교의 모든 학생을 조사하는 방식이 이에 해당합니다. 모집단을 층으로 나눈 후 각각에서 표본을 추출하는 ‘층화추출법’과는 차이가 있습니다.

4. 단순무작위추출법
: 모집단의 모든 개체가 동일한 확률로 선택될 수 있도록 하는 가장 기본적인 표본추출법입니다. 난수표, 컴퓨터 프로그램 등을 이용하여 완전히 무작위로 표본을 선정하는 방식입니다. 표본의 대표성이 보장되지만, 모집단이 크면 전체적인 조사 과정이 복잡해질 수 있습니다.

데이터를 측정할 때, 명목 척도는 이름, 성별, 지역 등과 같은 범주형 데이터를 구분하는 데 사용됩니다. 이는 데이터의 속성을 구분하기 위한 척도로, 순서나 크기를 나타내지 않습니다.

2. 서열 척도
: 서열 척도는 데이터 간의 순서를 나타내지만, 크기나 간격을 나타내지 않습니다. 이는 순위나 등급을 매길 때 사용되며, 이름, 성별, 지역과 같은 데이터에는 적합하지 않습니다.

3. 등간 척도
: 등간 척도는 데이터 간의 간격을 나타내지만, 절대적인 0점이 없습니다. 이는 온도나 시간과 같은 데이터를 측정할 때 사용되며, 범주형 데이터에는 적합하지 않습니다.

4. 비율 척도
: 비율 척도는 데이터 간의 절대적인 크기와 간격을 나타내며, 절대적인 0점이 존재합니다. 이는 길이, 무게, 시간과 같은 데이터를 측정할 때 사용되며, 범주형 데이터에는 적합하지 않습니다.

4. 추론 통계: 표본을 통해 모집단의 특성을 추론하는 통계적 방법입니다. 이는 모집단의 모수를 추정하거나 가설 검정을 수행하는 데 사용됩니다.

5. 확률변수 X가 확률질량함수 f(x)를 갖는 이산형 확률변수인 경우 그 기댓값으로 옳은 식을 고르시오.

정답보기
정답
1 E(X)=∑xf(x)
해설
이산형 확률변수의 기댓값은 확률변수 x의 값과 그 값이 발생할 확률 f(x)의 곱을 모든 가능한 값에 대해 합산하여 계산한다.
비기봇에게 문의중…
비기봇 해설
기댓값은 확률변수가 가질 수 있는 모든 값의 평균적인 크기를 나타내는 지표입니다. 이산형과 연속형 확률변수에 따라 정의 방식이 달라집니다.

1. E(X)=∑xf(x)
: 이 식은 이산형 확률변수의 기댓값을 계산하는 공식입니다. 확률변수 X의 값 x와 해당 값이 발생할 확률 f(x)를 곱한 후 합산하는 방식으로 정의되므로, 문제의 정답입니다.

2. E(X)=∫xf(x)dx
: 이 식은 연속형 확률변수의 기댓값을 계산할 때 사용됩니다. 연속형 확률변수에서는 확률질량함수 대신 확률밀도함수(PDF)가 사용되며, 적분을 통해 기대값을 구합니다.

3. E(X)=∑x2f(x)
: 이 식은 기댓값의 정의가 아니라, 분산 계산 시 사용되는 제곱값의 기댓값을 나타냅니다. 이 식은 분산을 구할 때 다음과 같이 활용됩니다.
Var(X)=E(X2)−(E(X))2

4. E(X)=∫x2f(x)dx
: 이 식은 연속형 확률변수에서 제곱값의 기댓값을 계산할 때 사용됩니다. 연속형 확률변수에 적용되며, 문제에서 묻는 이산형 확률변수의 기댓값 계산과는 관련이 없습니다.

비기봇 해설
앞면이 나올 확률이 0.5인 동전을 3번 던졌을 때, 앞면이 한 번만 나올 확률을 구하기 위해 이항 확률을 사용할 수 있습니다.

1. 주어진 조건 확인
이항 분포의 확률 질량 함수(PMF)는 아래와 같습니다.
P(X=k)=(nk)pkqn−k
이때 (nk)는 이항계수를 나타내며, (nk)=n!k!(n−k)! 입니다.

• p=0.5: 앞면이 나올 확률(=성공 확률)
• q=1-p=0.5: 뒷면이 나올 확률(=실패 확률)
• n=3: 시도 횟수
• k=1: 앞면에 나오는 횟수(= 성공 횟수)

2. 계산
• 이항계수 계산
(31)=3!1!(3−1)!=3×2×11×(2×1)=3

• 확률 계산
P(X=1)=(31)(0.5)1(0.5)3−1=3×(0.5)1times(0.5)2=3×12×14=3×18=38

따라서, 동전을 3번 던졌을 때 앞면이 한 번만 나올 확률은 3. 38입니다.

해설
독립 사건의 경우 조건부 확률 P(B∣A)는 P(B와 동일하다. 따라서, P(B∣A)=P(B)=0.4로 ③이 정답이 된다.
비기봇에게 문의중…
비기봇 해설
두 사건 A와 B가 독립일 경우, 조건부 확률 P(B|A)는 P(B)와 동일합니다. 이는 독립 사건의 특성으로, 한 사건의 발생이 다른 사건의 발생에 영향을 주지 않기 때문입니다. P(B|A) = P(B)임을 증명하는 과정은 아래와 같습니다.

조건부 확률은 아래와 같이 정의됩니다.
P(B|A)=P(A∩B)P(A)

독립 조건은 아래와 같습니다.
P(A∩B)=P(A)⋅P(B)

따라서, P(B|A)=P(B) 입니다.

주어진 조건 P(A) = 0.3, P(B) = 0.4로 풀이해보면, 독립 사건의 정의에 따라 P(B|A) = P(B)가 되고, P(B) = 0.4 = P(B|A)가 됩니다.

비기봇에게 문의중…
비기봇 해설
확률 및 확률분포는 사건과 확률 변수의 특성을 설명하는 데 사용됩니다. 이산형 확률분포는 이산적인 값만을 가질 때, 연속형 확률분포는 실수 범위의 값을 가질 때 사용됩니다.

1. 모든 사건의 확률값은 0 이상 1 이하이다.
: 확률의 기본 규칙에 따르면, 모든 사건의 확률은 0에서 1 사이의 값을 가져야 합니다. 0은 사건이 절대 발생하지 않음을, 1은 사건이 항상 발생함을 의미합니다. 예를 들어, 동전을 던질 때 "앞면이 나올 확률"은 0.5로, 0과 1 사이의 값입니다. 이 설명은 확률의 기본 정의를 반영하며 적절합니다.

2. 서로 배반인 사건들의 합집합의 확률은 각 사건의 확률을 더한 값과 같다.
: 배반 사건은 동시에 발생할 수 없는 사건을 의미합니다. 예를 들어, 주사위를 던질 때 "1이 나오는 사건"과 "2가 나오는 사건"은 배반 사건입니다. 이러한 경우, 합집합의 확률은 개별 사건 확률의 합으로 계산됩니다. 즉, P(1 또는 2) = P(1) + P(2)입니다.

3. 두 사건 A와 B가 독립이면, P(B∣A)=P(B)가 성립한다.
: 독립 사건은 한 사건의 발생이 다른 사건의 확률에 영향을 미치지 않는 상황을 의미합니다. 따라서, 독립 사건의 경우 P(B|A) = P(B)가 성립합니다. 예를 들어, 동전을 던지는 사건과 주사위를 던지는 사건은 서로 독립적이므로, 동전의 결과가 주사위 확률에 영향을 미치지 않습니다.

4. 확률변수가 연속적인 값을 가지는 경우를 이산형 확률분포라 한다.
: 확률변수가 연속적인 값을 가지는 경우는 연속형 확률분포로 설명됩니다. 연속형 확률분포에서는 확률 밀도 함수(PDF)를 사용하여 특정 구간에 대한 확률을 계산하며, 특정 값 자체의 확률은 0입니다. 반면, 이산형 확률분포는 확률변수가 특정 값(예: 정수)을 가지는 경우에 사용됩니다. 이 설명은 연속형과 이산형의 개념을 혼동하였으므로 부적절합니다.

비기봇 해설
1) 제1종 오류(Type I Error)와 α(알파)의 의미
• 제1종 오류는 귀무가설(H0)이 참인데도 이를 기각하는 오류입니다.
• 제1종 오류가 발생할 확률은 유의수준(α)입니다.
• 즉, 우리가 미리 설정하는 기준값(예: 0.05, 0.01 등)이며, 실제 제1종 오류가 발생할 확률을 의미합니다.

2) p-value의 의미와 차이점
• p-value는 귀무가설이 참이라는 가정 하에서, 현재 관측된 데이터보다 더 극단적인 데이터가 나올 확률입니다.
• p-value가 α보다 작으면 귀무가설을 기각하지만, p-value 자체가 제1종 오류의 확률을 의미하는 것은 아닙니다.
• 즉, p-value는 실험 결과에 따른 확률값일 뿐, 제1종 오류를 범할 확률과는 개념이 다릅니다.

1. α(알파)
: 제1종 오류를 범할 실제 확률을 나타내는 값은 유의수준(α)입니다. 연구자가 사전에 설정한 값으로, 귀무가설이 참일 때 이를 기각할 확률을 의미합니다.

2. p-value
: p-value는 귀무가설이 참일 때, 현재 데이터보다 더 극단적인 데이터가 나올 확률입니다. p-value는 실험 결과에 따라 변하는 값이지, 제1종 오류의 실제 확률이 아닙니다. 제1종 오류의 실제 확률은 사전에 설정된 유의수준(α) 값입니다.

3. 검정통계량
: 검정통계량(Test Statistic)은 데이터와 귀무가설 간의 차이를 수치적으로 나타내는 값입니다. 제1종 오류의 확률과 직접적인 관련이 없습니다. 예를 들어, Z검정에서는 Z값, t-검정에서는 t값이 검정통계량에 해당합니다.

4. 1-α
: 1-α는 제1종 오류를 범하지 않을 확률(즉, 신뢰수준)을 의미합니다. 이는 귀무가설이 기각되지 않을 확률이며, 귀무가설이 참일 확률이 아닙니다. 예를 들어, α=0.05라면 신뢰수준은 1-0.05 = 0.95(95%)로, 귀무가설이 기각되지 않을 확률을 나타냅니다.

1. F-statistics
: F-통계량은 모형 전체의 유의성을 평가하는 데 사용됩니다. 모형이 설명력을 가지는지 검정하기 위해, 모형에 의한 변동(Explained Variance)과 잔차에 의한 변동(Unexplained Variance) 을 비교하여 계산합니다. F-통계량이 크고, P-value가 유의수준(α)보다 작으면, 모형 전체가 종속변수를 잘 설명하고 있음을 의미합니다.

2. T-statistics
: T-통계량은 개별 독립변수가 종속변수에 유의미한 영향을 미치는지 확인하는 데 사용됩니다. 즉, 회귀모형의 전체 유의성을 평가하지 않고, 특정 변수 하나가 중요한지를 검정합니다. 모형 전체의 유의성을 평가하기에는 적합하지 않습니다.

3. P-value
: P-value는 F-통계량 또는 T-통계량의 결과를 해석할 때 사용됩니다. P-value는 검정 통계량의 결과값이 귀무가설을 기각할 수 있는 수준인지를 판단하는 기준입니다. 그러나 P-value는 독립적인 통계량이 아니라 검정 결과의 해석 도구입니다.

4. AIC(Akaike Information Criterion)
: AIC는 모형의 적합도와 복잡성을 동시에 고려하여 모형 간의 비교를 수행하는 지표입니다. AIC는 모형의 유의성을 평가하기보다는 모형 간 최적화를 위한 지표로 사용되며, 모형 전체의 유의성 평가와는 직접적인 관련이 없습니다.

: 표준화 회귀 계수(Standardized Coefficients, β)는 독립변수 간 단위 차이를 제거한 상태에서 상대적 중요도를 비교하는 데 사용됩니다. 하지만 p-value를 고려하지 않으면 통계적으로 유의하지 않은 변수도 중요하다고 판단할 가능성이 있어, 가장 중요한 변수 선정 기준으로는 부적절합니다.

: 원 자료(raw data)로 추정한 계수(회귀 계수, unstandardized coefficient)는 변수의 단위에 따라 다르게 표현됩니다. 예를 들어, 키(cm)가 종속변수일 때, 독립변수 몸무게(kg)의 회귀 계수는 0.5일 수도 있고, 소득(원)의 회귀 계수는 100000일 수도 있습니다. 단위가 다르므로 계수 크기만으로 중요도를 비교할 수 없으며, 단위가 다르면 계수 크기 비교가 무의미하므로, 표준화된 β를 기준으로 보는 것이 적절합니다.

정상요인은 데이터가 시간의 흐름에 따라 일정한 통계적 특성을 유지하는 것을 의미하며 시계열 분해의 구성 요소에 포함되지 않는다. 계절요인, 추세요인, 순환요인은 시계열 데이터를 분석하고 분해할 때 주요한 구성 요소로 사용된다.

비기봇 해설
시계열 데이터 분석에서는 데이터를 분해하여 주요 패턴을 이해하는 것이 중요합니다. 일반적으로 시계열 데이터는 추세, 계절, 순환 요인으로 구성되며, 정상요인은 포함되지 않습니다.

1. 정상요인
: 정상요인은 시계열 데이터에서 특정한 패턴을 말하는 것이 아니라, 시계열이 평균과 분산이 시간에 따라 일정하게 유지되는 정상성 여부를 평가하는 개념입니다. 이는 시계열 분해에서 구성 요소로 포함되지 않습니다.

2. 계절요인
: 계절요인은 데이터가 일정한 주기로 반복되는 변화를 나타냅니다. 예를 들어, 월별 판매량이 특정 시즌에 반복적으로 증가하거나 감소하는 경우가 이에 해당합니다. 이는 시계열 분해의 중요한 구성 요소 중 하나입니다.

3. 추세요인
: 추세요인은 데이터의 장기적인 증가나 감소 추세를 나타냅니다. 예를 들어, 회사의 연간 매출이 꾸준히 증가하는 경향을 보이는 경우, 이는 추세요인으로 설명할 수 있습니다.

4. 순환요인
: 순환요인은 데이터가 장기적인 경제 또는 비즈니스 사이클과 같은 요인에 의해 변동하는 패턴을 의미합니다. 이는 계절성보다 긴 주기로 반복되는 경향을 분석할 때 사용됩니다.

시계열 분석에서 자기 자신의 과거값을 사용하여 설명하는 모형은 AR(자기회귀 모형)입니다. 이는 과거의 데이터를 기반으로 현재의 값을 예측하는 방식으로, 자기 자신의 과거값을 사용하여 현재값을 설명합니다.

1. AR (자기회귀 모형)
: 보기의 설명은 AR 모형의 특징을 정확히 묘사하고 있습니다. AR 모형은 시계열 데이터의 현재값을 과거값의 선형 결합으로 표현하며, 백색 잡음도 포함됩니다. p차 모형은 과거 최대 p시점까지의 데이터를 포함합니다.

2. MA (이동 평균 모형)
: MA 모형은 시계열의 현재값을 백색 잡음(과거의 오차항)의 선형 결합으로 표현합니다. MA 모형에서는 자기 자신의 과거값이 아닌 과거의 예측 오차를 기반으로 현재값을 설명합니다. 따라서 보기와는 맞지 않습니다.

3. ARMA (자기회귀이동평균 모형)
: ARMA 모형은 AR 모형과 MA 모형을 결합한 형태로, 과거값(AR)과 과거의 오차(MA)를 동시에 사용하여 데이터를 설명합니다. 하지만 보기는 AR 모형의 설명에 해당하므로 정답이 아닙니다.

4. ARIMA (자기회귀누적이동평균 모형)
: ARIMA 모형은 비정상 시계열 데이터를 대상으로 차분(differencing)을 통해 정상성을 확보한 후 ARMA 모형을 적용하는 방법입니다. 정상 시계열 모형인 AR 모형의 설명과는 다릅니다.
